## Contributing to Doc-Hunt In The Nutshell
You can do these steps in order to contribute to Doc-Hunt:
* Fork the repository.
* Clone the repository.
* Create a new branch.
* Add your changes.
* Commit your changes.
* Push your changes to your branch.
* Create a pull request WITH EMOJI.
